BioDigital

  • Big, high-detail anatomy visualizations
  • Uses:
    • Education
    • Training
    • Marketing

BioDigital

  • Clients frequently want to visualize
    • Fluid moving through vessels
    • Molecular interactions
    • Cellular activity

Particle System

  • These effects can be efficient rendered with a particle simulation
    • Draw points
    • Fine-grained simulation components to apply forces/constraints

Before the Particle System

  • Particles would be rendered as low-res spheres
  • Each particle would be manually animated by an artist

Requirements for the Particle System

  • Maximize artist control
  • Acturately represent anatomical phenomenon

Architecture

  • Particles are simply separate arrays of properties
    • Position
    • Velocity
    • Color
    • Age

Architecture

  • Particles updated each frame by components
    • Creators: Initialize a property
    • Updater: Update a property
    • Constraint: Ensure particle propery remains within bounds
  • Each must be minimal to allow artists to combin as desired

    • Updates Over Time

    • Initially pseudo-physical simulation of colored points

    Updates Over Time

    • Added sprite texturing
      • Significantly improved representational quality

    Updates Over Time

    • Artists can animated system components over time
