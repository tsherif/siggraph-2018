BioDigital

BioDigital

  • Big, high-detail anatomy visualizations
  • Uses:
    • Education
    • Training
    • Marketing

BioDigital

  • Clients frequently want to visualize
    • Fluid moving through vessels
    • Molecular interactions
    • Cellular activity

Particle System

  • These effects can be efficiently rendered with a particle simulation
    • Draw points
    • Fine-grained simulation components to apply forces/constraints

Before the Particle System

  • Particles would be rendered as low-res spheres
  • Each particle would be manually animated by an artist

Requirements for the Particle System

  • Maximize artist control
  • Acturately represent anatomical phenomenon

Architecture

  • Particles are simply separate arrays of properties
    • Position
    • Velocity
    • Color
    • Age

Architecture

  • Particles updated each frame by components
    • Creators: Initialize a property
    • Updater: Update a property each frame
    • Constraint: Ensure property remains within bounds

Example Creator

        
          // Initialize position to somewhere in defined ellipsoid
          {
            type: "SpherePosition",
            scale: [5, 10, 5],
            translate: [2, 4, 3],
            rotate: [20, 30, 0]
          }

Example Updater

        
          // Apply force towards/away from given point
          {
            type: "RadialForce",
            positions: [5, 10, 5],
            force: 10,
            shape: {
              type: Sphere",
              scale: [3, 4, 3]
            }
          }

Example Constraint

        
          // Reflect particle velocity if it crosses the plane
         {
             type: "ShapeBounce",
             shape: {
                 type: "Plane",
                 rotate: [0, 0, -45],
                 translate: [-10, -10, 0],
                 direction: "in"
             }
         }

Architecture

  • Each component has a minimal function in the sim
  • Allows artists to combine them in expressive ways

Applications

  • Initially limited to basic processes and ambient effects

Applications

  • Over time added features that significantly increased quality
    • Texturing
    • Animate component parameters over time
